# Samples

Anonymised example outputs showing what the tool produces.

## Files

| File | Description |
|------|-------------|
| `sample_batch_jobs.csv` | Example batch input with three input types (URL, pasted text, file path) |
| `sample_applications.json` | Example tracker output with fake company and personal data |

## Note

All sample data is fictional. No real resume content, contact information, or application data is included here.

Real output files (`*.docx`) are gitignored by default to protect private data.
To include a sample DOCX, generate one using entirely fake data and add it to this folder.
